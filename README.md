## Application Features

### For Non-Authorized Users

- Browse word boards
- View words using the "Skip" button

<img src="https://github.com/user-attachments/assets/13cb2370-4acb-4758-91c6-d4f54fcc88cd" width="400" alt="Word Boards">
<img src="https://github.com/user-attachments/assets/3b80d1bc-3529-4b6b-b446-e55dd30d9274" width="400" alt="Word Viewing">

### For Authorized Users

- Add words to personal dictionary
- Rate word knowledge across 4 levels:
  - Don't know
  - Hard
  - Good
  - Easy
- Spaced repetition system
- Sort words by mastery level with color indication
